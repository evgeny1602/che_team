БД
    USER_GROUP
        id
        name  (Игроки, Админы, ...)
    USER
        id
        name  (Иван Иванов, Петр, ... )
        tg_id
        tg_username
        user_group_id
        is_approved
    EVENT
        id
        date_time    
        duration    
        name  (Тренировка в Юбилейном, Тренировка в РИИ с тренером, ...)
        description     
        min_users_count   
        max_users_count
        price
        user_id  
        is_held (true = игра состоялась, false = игра не состоялась)
    USER_TO_EVENT
        id
        event_id
        user_id
        friends_count
        result  (visited, missed)
